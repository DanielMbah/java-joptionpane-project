# Java Pay Calculator

This project demonstrates the use of JOptionPane for graphical user input and output in Java.

The program:
- Prompts the user for hourly pay rate and hours worked
- Calculates gross pay using a value-returning method
- Calculates tax (15%) using a separate method
- Displays gross pay, tax amount, and net pay in a dialog box

## Concepts Used
- Static methods
- Method calls
- Parameter passing
- Value-returning methods
- Percentage calculations
- JOptionPane GUI dialogs
